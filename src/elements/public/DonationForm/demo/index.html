Foxy Donation form demonstration

The simplest form: a button

Here's the code:

<donation-form storeSubdomain="MyFoxySubdomain" value="100"></donation-form>

Here's the result:

Offer some choices of values

You can offer some value choices by simply adding the "valueOptions" attribute.

Here's the code:

<donation-form storeSubdomain="MyFoxySubdomain" valueOptions="[10, 30, 50, 100]"></donation-form>

Here's the result:

Allow the donner to set a custom ammount

You can use the "askValueOther" attribute to add a "other" field

Here's the code:

      <donation-form storeSubdomain="MyFoxySubdomain"
                     valueOptions="[10, 30, 50, 100]"
                     askValueOther="true"
                     ></donation-form>

Here's the result:

Allow the user to choose a designation for the donation

You can use the "designationOptions" to allow for specifying a designation.

Here's the code:

      <donation-form storeSubdomain="MyFoxySubdomain"
                     valueOptions="[10, 30, 50, 100]"
                     designationOptions='["Rebuild the School", "Medical Assistance", "Psicological Assistance", "Daily Meals"]'
                     askValueOther="true"
                     askDesignationOther="true"
                     ></donation-form>

Here's the result:

Change order and type

You can change the field type used. Instead of using radio buttons or check boxes, you may prefer to use select boxes instead.

You can also change the order of any part of the form. To do that, add a "Weight" suffix to the name of the part of the form you want to set a weight to to compose an attribute. The greater the weight the further down that part will be placed.

Notice in the previous example that designation after the value by default. To invert that do the following:

      <donation-form storeSubdomain="MyFoxySubdomain"
                     valueOptions="[10, 30, 50, 100]"
                     designationOptions='["Rebuild the School", "Medical Assistance", "Psicological Assistance", "Daily Meals"]'
                     askValueOther="true"
                     askDesignationOther="true"
                     valueType="select"
                     designationType="select"
                     designationWeight="1"
                     valueWeight="2"
                     ></donation-form>

Include comment and anonymous donation button

On top of all that you may allow the donner to ask to remain anonymous and to leave a comment.

Please notice that the "anonymous" options only means that the donner does not wish to be publicly recognized.

Here's the code:

      <donation-form
        currency="€;"
        askComment
        askAnonymous
        valueOptions="[ 20, 100, 250]" >
      </donation-form>

Include your own markup

You can include whatever markup you wish above with your form parts. Simply use the name of that specific part as the slot where to place it.

Here's the code:

      <donation-form currency="â‚¬" valueOptions="[1, 2, 3]"
        askValueOther
        askComment
        askAnonymous
        askDesignation
        commentWeight="8"
        anonymousWeight="0"
        valueWeight="8"
        valueType="select"
        >
        <p slot="value">
          You have a crucial role in our mission. None of this could have been
          done without your generous donation.
          <img src="/dev/img/tree.jpg" alt="A nice tree icon.">
        </p>
        <strong slot="anonymous">Would you like to remain anonymous?</strong>
        <p slot="comment">Would you like to leave us a message?</p>
      </donation-form>

Add your own markup before or after any form part

If you want to add a markup before or after any given part, you can use the "before-0" and "after-0" slot names, where 0 stands for the position of the form part.

Here's the code:

      <donation-form askValueOther askComment askAnonymous askDesignation >
        <div slot="after-0">After 0</div>
        <div slot="after-1">After 1</div>
      </donation-form>

Of course, add your own fields

You can add your own fields to the form. HTML elements will be added to the form element so that they can be send together with the form.

Here is the code:

      <donation-form designationOptions='["rescue", "medical treatment", "reintegration in nature"]' askComment askAnonymous >
        <div slot="after-1" class="any class name you like">
          <label for="branch">Choose a branch</label>
          <select id="branch" name="branch">
            <option value="main">Main</option>
            <option value="south">South Branch</option>
            <option value="north">North Branch</option>
            <option value="west">West Branch</option>
            <option value="east">East Branch</option>
          </select>
        </div>
        <div slot="after-2" class="my-field">
          <label>Leave a percentage to administration</label>
          <p>If you choose to leave a percentage to administration, this ammount can be used to internal administrative activities and to help raise more funds.</p>
          <input type="number"
                 step="1"
                 min="0"
                 value="0"
                 max="100"></input>
        </div>
      </donation-form>

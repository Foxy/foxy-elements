# Collection Items

Custom element adding infinite scroll functionality to another custom element representing a hypermedia resource within a hypermedia collection. Once added to the document and configured, it will track scroll position and load new pages of the collection when appropriate. Collection items are rendered as custom elements with the following attributes:

- **lang**: language choice, propagated from `<foxy-collection-pages>`;
- **href**: collection page URL;

Resource elements are expected to comminicate with the API using `request` events, just like the rest of the Foxy Elements,

## Example: Collection Items + Attribute Card

The code below will be loading and displaying all attributes of the store in chunks of 20 attributes per page using the `<foxy-attribute-card>` and `<foxy-spinner>` elements.

```html
<foxy-collection-items
  element="foxy-attribute-card"
  spinner="foxy-spinner"
  first="https://api.foxy.test/stores/8/attributes"
  lang="en"
>
</foxy-collection-items>
```

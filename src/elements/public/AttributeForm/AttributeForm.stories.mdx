# Attribute

[![hAPI](https://img.shields.io/badge/hAPI-attribute-green.svg)](https://api.foxycart.com/rels/attribute)

## States

Below are all the states this component can render.

### Loading

Displayed while loading the resource.

<Canvas>
  <Story name="Loading">
    {() =>
      html`
        <foxy-attribute-form href="https://demo.foxycart.com/s/admin/sleep"> </foxy-attribute-form>
      `
    }
  </Story>
</Canvas>

### Error

Displayed when resource load fails.

<Canvas>
  <Story name="Error">
    {() =>
      html`
        <foxy-attribute-form
          href="https://demo.foxycart.com/s/admin/customer_attributes/not-found"
        ></foxy-attribute-form>
      `
    }
  </Story>
</Canvas>

### Empty

Displayed when there's no `href` attribute.

<Canvas>
  <Story name="Empty">
    {() =>
      html`
        <foxy-attribute-form
          parent="https://demo.foxycart.com/s/admin/customers/0/attributes"
        ></foxy-attribute-form>
      `
    }
  </Story>
</Canvas>

### Ready

Displayed after the resource is loaded.

<Canvas>
  <Story name="Ready">
    {() =>
      html`<foxy-attribute-form
        href="https://demo.foxycart.com/s/admin/customer_attributes/0"
      ></foxy-attribute-form>`
    }
  </Story>
</Canvas>

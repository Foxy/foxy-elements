# foxy-payment-method-selector

Payment method selector element that renders available checkout payment options, tracks the selected option, and tokenizes the selected method payload.

Required attributes: none. The element still requires checkout client state to be initialized before it can render payment options and tokenize.

If checkout state is initialized but exposes no recognized payment methods, the element renders an unavailable state instead of fabricating a fallback option. In that state, <code>tokenize()</code> rejects because no payment method is selected.

## Tokenization Payload

<code>tokenize()</code> resolves with this payload. The
<code>tokenizationsuccess</code> event emits the same object as
<code>event.detail.payload</code>.

```ts
type PaymentMethodSelectorTokenizePayload = {
  optionIndex: number;
  optionType:
    | "saved-card"
    | "new-card"
    | "ach"
    | "stripe-card-element"
    | "stripe-payment-element"
    | "apple-pay"
    | "google-pay"
    | "generic";
  billingAddress?: {
    useShippingAddress: boolean;
    values: Record<string, string>;
  };
  token?: string;
  requestId?: string;
  paymentMethodId?: string;
  paymentMethodType?: string;
  cardBrand?: string;
  last4?: string;
  expirationMonth?: number;
  expirationYear?: number;
};
```

Every successful payload includes <code>optionIndex</code> and <code>optionType</code>. <code>billingAddress</code> is included after the billing address UI has reported values for the selected option.

## Usage Example

```html
<foxy-payment-method-selector
  id="payment-selector"
  lang="en-US"
  option-index="0"
  theme-primary="#111827"
  theme-radius="0.75rem"
></foxy-payment-method-selector>

<button type="button" id="selector-tokenize">Tokenize Selection</button>

<script type="module">
  import "https://cdn-js.foxy.io/elements@2/foxy-payment-method-selector.js";
  import { client as checkoutClient } from "@foxy.io/sdk/checkout/client";

  const selector = document.querySelector("#payment-selector");
  const selectorTokenizeButton = document.querySelector("#selector-tokenize");

  await checkoutClient.refresh();

  selector.addEventListener("tokenizationsuccess", (event) => {
    console.log("Payment method tokenized", event.detail.payload);
  });

  selector.addEventListener("tokenizationerror", (event) => {
    console.error("Payment method tokenization failed", event.detail.error);
  });

  selectorTokenizeButton.addEventListener("click", async () => {
    const payload = await selector.tokenize();
    console.log("Selector tokenize() result", payload);
  });
</script>
```
